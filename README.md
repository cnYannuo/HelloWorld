# 在 Cursor 编辑器中设置 Codex 模型

下面是一个在 Cursor 编辑器中启用并使用 Codex 模型的通用指南。不同版本的 Cursor 可能在界面文案上略有差异，但整体流程一致。

## 前置条件

- 已安装并登录 Cursor（需要可用的账号权限）。
- 有可用的模型配额/订阅（由账号决定）。

## 设置步骤

1. **打开设置**
   - 使用快捷键 `Ctrl+,`（macOS 为 `Cmd+,`），或从菜单进入 **Settings**。

2. **进入模型设置**
   - 在设置搜索框输入 `model` 或 `AI`，找到 **AI / Models**（或类似名称）的配置区域。

3. **选择 Codex 作为默认模型**
   - 在 **Default Chat Model** 或 **Default Edit Model** 中选择 `Codex`。
   - 如果你需要在不同场景切换模型，可分别设置 Chat 与 Edit 的默认模型。

4. **保存并验证**
   - 关闭设置面板。
   - 打开 Chat 或 Inline Edit，确认顶部模型标签显示为 `Codex`。

## 常见问题

- **找不到 Codex 选项**
  - 可能是账号权限或订阅未开通，或 Cursor 版本未更新。
  - 尝试更新 Cursor 到最新版本并重新登录。

- **模型切换后无变化**
  - 重启 Cursor；
  - 或在当前对话顶部的模型下拉菜单里手动切换。

## 快速检查清单

- [ ] Cursor 已更新到最新版本
- [ ] 已登录正确账号并有模型权限
- [ ] 设置中选择了 `Codex`
- [ ] Chat/Inline Edit 顶部显示 `Codex`
